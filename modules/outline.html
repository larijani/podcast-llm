podcast_llm.outline

Podcast outline generation module.

This module provides functionality for generating and structuring podcast outlines. It defines data models for representing podcast sections and subsections, and includes utilities for formatting and manipulating outline structures.

The module uses Pydantic models to enforce data validation and provide a clean interface for working with podcast outline components. The models support hierarchical organization of content with sections containing subsections, and include helper methods for string formatting.

Classes:

PodcastSubsection: Models an individual subsection within a podcast section PodcastSection: Models a major section containing multiple subsections

Example

section = PodcastSection(

title=”Main Discussion”, subsections=[

PodcastSubsection(title=”Key Concepts”), PodcastSubsection(title=”Historical Context”)

]

) print(section.as_str)
